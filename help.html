MDPD - Microbiome Database of Pulmonary Diseases

Home Browse Statistics Help Team

  1. The Search Option on the home page allows users to query the database with specific keywords for Disease, Assay type, Country, Isolation source, Instrument, Year.

    1. The search result page provides the information of the Run ID, BioProject ID, SRA Study ID, Disease, Disease Sub-group, Isolation Source, Instrument, Assay Type, Country and Year.

    2. The Run page gives the information of the Experiment ID, BioSample ID, Average Spot Length, Number of Bases, Center Name, Biome, and Therapeutics. The link beneath the title leads to NCBI SRA page and BioProject ID links to BioProject page in MDPD.

    3. The BioProject page shows the information related to the study and the total number of runs of the BioProject. If subgroups are present for the study, then a plot of differential taxa can be accessed by LDA plot. Taxonomic profile can be visualized with Krona plots for group or subgroup. The Run ID (s) link to the Run page and the link just beneath the title leads to the NCBI BioProject page.

    4. The LDA (Linear Discriminant Analysis) plot provides information about the differential biomarkers (genus for amplicon and species for WMS). While the bars denote the marker genus/species, the length depicts the log10 (LDA score) (log10 LDA ≥ 2).

    5. The taxonomic profile of microbiome data of different isolation sources such as sputum, BAL, Stool can be visualized by Krona plots. Two types of krona plots were made, (i) by merging all the runs and (ii) by keeping individual runs. The taxonomic level changes from lower (genus/species) to higher level (domain) as we move from the outer ring to the inner ring of the plot.
      For a better view, see the plot at "max depth = 6" for amplicon (amplicon) and "max depth = 7" for WMS (species).

  2. The browse page of MDPD is divided into four parts.

    1. Disease-wise BioProjects provide the information about the studies associated with the disease.

      Click any one of the diseases to view the list of BioProjects associated with the study. Click one BioProject to go to the BioProject page.

    2. Disease-wise taxonomic profile (Krona plot) across isolation sources help users to view the taxonomic profile of microbiome across different isolation sources of the lung diseases and healthy group using krona plot.

    3. Disease-wise similarity analysis across Gut-Lung biomes enables the users to fetch the list of shared prevalent microbes between lung and gut microbiome of each lung disease and healthy group. Isolation sources with the highest number of runs were taken for the analysis such as "sputum" and "stool" for "Asthma". Click "get result" to view the shared prevalent microbes.

      The prevalent taxa that are unique to lung, unique to gut biome and shared by both biome can be visualized with UpSet plot. The shared prevalent taxa list can be seen by diverging bar plot (Likert). A cut-off of minimum prevalence of 50% at detection level ≥ 0.0001 was taken to obtain the prevalent taxa.

    4. Differential analyses of disease pairs (LDA plot and heatmap) provides the facility to obtain the set of differentially abundant taxa between a pair of lung disease or disease- healthy group. The analysis was done on the isolation sources that are available for both conditions. Click any button "Show" to view the associated pair.

      Users can obtain the LDA plot (horizontal bar plot) by clicking the "Get LDA plot" to see the differentially abundant taxa for the pair of lung conditions/group (cut-off log10 LDA ≥ 2). With the "Get heatmap" option users can view the relative abundances of differentially abundant taxa (cut-off log10 LDA ≥ 3) for the disease pair and the BioProjects as the source of marker taxa.

      There are few buttons to modify the plot.
